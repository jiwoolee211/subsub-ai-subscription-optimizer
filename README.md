# SUBSUB AI 기반 구독 소비 최적화 서비스 멋쟁이사자처럼 14기 아이디어톤 본선 진출 ![main](assets/main_ui.png) ---

## 🚀 Overview 사용자 행동 데이터를 기반으로 '낭비되는 구독'을 탐지하고 최적화하는 AI 서비스 --- 

## ❗ Problem - 사용하지 않는 구독 유지 경험 (66.2%) - 무료체험 후 자동결제 경험 (76.9%) 👉 문제 핵심: 결제 중심 관리 → **사용 기반 판단 부재** --- 

## 💡 Solution - 사용 패턴 기반 구독 분석 - 방치 구독 자동 감지 - 해지 / 요금제 변경 추천 --- 

## 🔧 Key Features ### 1) AI 구독 레포트 ![report](assets/report_ui.png) - 사용 시간 / 빈도 / 간격 분석 - 구독 유형 분류 (집중형 / 낭비형 등) - 개인화된 해지 및 요금제 추천 ---

### 2) 통합 알림 시스템 ![noti](assets/notification_ui.png) - 결제일 사전 알림 - 무료체험 종료 알림 - 사용량 감소 감지 --- 

## 🔄 Service Flow 서비스는 사용자 데이터 분석을 중심으로 구독 플랫폼 및 외부 파트너와 연결됩니다. ![flow](assets/flow.png) --- 

## 🤖 AI Architecture - K-Means → 사용자 패턴 군집화 - Isolation Forest → 방치 구독 탐지 - Recommendation → 유지 / 해지 / 변경 추천 👉 핵심: **사용 데이터 기반 의사결정** --- 

## 🏗 Tech Stack (Planned) - Frontend: React Native, TypeScript - Backend: Django - Database: MySQL - AI/ML: Python (scikit-learn) --- 

## 📊 Outcome - 멋쟁이사자처럼 14기 아이디어톤 본선 진출 - 데이터 기반 서비스 기획 및 AI 설계 ---
